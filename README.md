A modern, responsive, and sleek portfolio website built to showcase my journey as a Software Developer. This project features a dark-themed UI, smooth navigation, and a structured layout for Skills, Education, and Contact information.

🛠️ Tech Stack
1. HTML5: Semantic structure and content.
2. CSS3: Custom styling, Flexbox, Grid, and Animations.
3. Font Awesome: For professional iconography.
4. Google Fonts: Utilizing 'Poppins' for a clean, modern look.

✨ Features
1. Smooth Scrolling: Integrated navigation that slides gracefully to each section.
  
2. Responsive Design: Fully optimized for Desktops, Tablets, and Mobile devices.
 
3. Interactive UI: Hover effects on social icons, skill cards, and education boxes.

4. Education Timeline: A clean, box-based layout to showcase academic milestones.
 
5. Contact Form: A styled interface for professional inquiries.
